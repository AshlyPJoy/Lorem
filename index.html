Easy to use, stylish placeholders

Easy to use, stylish placeholders Just add your desired image size (width & height)
after our URL, and you'll get a random image.

https://picsum.photos/200/300

To get a square image, just add the size.

https://picsum.photos/200/300
Specific Image

Get a specific image by adding /id/{image} to the start of the url.

https://picsum.photos/200/300

You can find a list of all the images here

Static Random Image

Get the same random image every time based on a seed, by adding /seed/{seed} to the start of the url.

https://picsum.photos/200/300
Grayscale

Get a grayscale image by appending ?grayscale to the start of the url.

https://picsum.photos/200/300
Blur

Get a blurred image by appending ?blur to the start of the url.

https://picsum.photos/200/300/?blur

You can adjust the amount of blur by providing a number between 1 and 10

https://picsum.photos/200/300/?blur
Advanced Usage

You may combine any of the options above. For example, to get a specific image that is grayscale and blurred.

https://picsum.photos/200/300

To request multiple images of the same size in your browser, add the random query param to prevent the images from being cached:

< img src="https://picsum.photos/200/300?random=1">

< img src="https://picsum.photos/200/300?random=2">

If you need a file ending, you can add .jpg to the end of the url.

https://picsum.photos/200/300.jpg

To get an image in the WebP format, you can add .webp to the end of the url.

https://picsum.photos/200/300.webp

List Images

Get a list of images by using the /v2/listendpoint.

https://picsum.photos/v2/list

The API will return 30 items per page by default. To request another page, use the ?page parameter. To change the amount of items per page, use the ?limitparameter.

https://picsum.photos/v2/list?page=2&limit=100

The Linkheader includes pagination information about the next/previous pages

Image Details

Get information about a specific image by using the the /id/{id}/info endpoint

https://picsum.photos/id/0/info

You can find out the ID of an image by looking at the Picsum-IDheader, or the User Comment
